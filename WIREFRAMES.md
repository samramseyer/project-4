## Design Notes

### Color Scheme (Hot Rod Theme)

- **Background**: Black (#1a1a1a, #2a2a2a)
- **Chrome/Silver**: (#c0c0c0, #e0e0e0, #ffffff)
- **Accent Red**: (#dc1414, #ff3333, #aa0000)
- **Text**: Chrome gray on dark backgrounds

### Typography

- **Headers**: Impact, Arial Black
- **Body**: Arial, sans-serif
- **Style**: Uppercase, increased letter-spacing (2-5px)

### Visual Effects

- Metallic linear gradients for chrome shine
- Red neon glow (text-shadow: 0 0 10px rgba(220,20,20,0.6))
- Inset shadows for depth
- 3D button effects with multiple box-shadows

### Key Features

1. **Left Sidebar Navigation** - Sticky, scrollable category menu
2. **Inline Validation** - Real-time error messages on registration
3. **Vote System** - Up/down arrows with vote counts
4. **Accepted Answers** - Green glow for solved questions
5. **Category Filtering** - Click categories to filter questions
6. **Chronological Sorting** - Questions sorted newest first by default
